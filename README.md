# ca-lite 
based on openssl and cryptopro CSP.

## Supported OS:
- Ubuntu 16.04
- Windows 10

## Usage: 
1. [Download](https://github.com/fullincome/ca-lite/releases/latest) latest version **ca-lite** (or build from source).  
2. If need test keys:
 - **./scripts/gen_key.sh**
 - **CryptoPro CSP**

## Build:  
Easy way to build ca: **./configure && make**.
